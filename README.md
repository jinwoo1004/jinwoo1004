## 원칙

**01 언어보다 문제를 먼저 본다**<br/>
스택은 도구입니다. 상황에 맞는 기술을 골라 빠르게 적용합니다.

**02 기능이 아니라 플랫폼을 설계한다**<br/>
인증과 연동, 데이터, 운영까지 공통 구조를 먼저 그립니다.

**03 데이터를 자산으로 남긴다**<br/>
오늘의 기록이 내일의 AI와 분석 기반이 됩니다.

**04 안정성과 소통으로 완결한다**<br/>
시큐어 코딩과 정합성 검증을 기본으로 현업과 반복 협업합니다.
